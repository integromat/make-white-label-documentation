# Change an organization's reset date and restart period

An organization's reset date and restart period define when and how often your Make instance automatically renews that organization's consumables. You can redefine these parameters and reset an organization's consumables at any time. For example, when your customer:

* Changes between monthly and yearly subscriptions.
* Upgrades to more features or operations.
* Downgrades their subscription.
