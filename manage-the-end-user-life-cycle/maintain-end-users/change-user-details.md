# Change user details

In case of an entry error or changes in your customer's situation, you can update their details. User details include the following parameters:

* Email
* Name
* Language
* Locale
* Timezone
* Country

Use the following API endpoint to update user details:

Update user details

{% hint style="info" %}
You can also edit the user's details in the Administration interface. Go to Administration > Users and select Change details from the drop menu in the top right-hand corner.
{% endhint %}
