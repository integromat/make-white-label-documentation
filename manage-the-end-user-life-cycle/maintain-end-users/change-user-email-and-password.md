# Change user email and password

When your end user loses or forgets their login credentials, use the following administrative API endpoints:

* Update a user's email address

{% hint style="info" %}
You can also edit the user's email address in the Administration interface. Go to **Administration > Users** and select Change details from the drop menu in the top right-hand corner.
{% endhint %}

* Reset a user's password

{% hint style="info" %}
You can also update the user's password in the Administration interface. Go to **Administration > Users** and select **Change password** from the drop menu in the top right-hand corner.
{% endhint %}
