# Facebook Conversions API connection update by May 9, 2026

**Action required:** **Reauthorize Facebook Conversions API connections**

Meta is updating its security requirements for Facebook Conversions API connections. To keep your scenarios running without interruption, you must reauthorize affected connections after this update is released and before **May 9, 2026**.

### What’s changing

Meta’s updated security requirements affect existing connections for the **Facebook Conversions API** and **Facebook Conversions API for CRM** apps.

After the update is released, these connections must be reauthorized to continue working. Your existing scenarios will continue to run until **May 9, 2026**. After this date, scenarios using connections that haven’t been reauthorized might fail.

### What you need to do

Reauthorize your affected Facebook Conversions API connections before **May 9, 2026**.

1. Go to the **Connections** page in your Make account.
2. Find your existing connection for **Facebook Conversions API** or **Facebook Conversions API for CRM**.
3. Click **Re-authorize**.
4. Follow the prompts to sign in to your Facebook account again.
